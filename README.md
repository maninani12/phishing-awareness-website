# Phishing Awareness Website

A web-based cybersecurity awareness project designed to educate users about phishing attacks and how to identify suspicious emails, links, and websites.

The platform demonstrates common phishing techniques and provides examples to help users recognize potential threats and improve their online security awareness.

## Features

- Educational content explaining phishing attacks
- Examples of phishing emails and websites
- Tips for identifying suspicious links and messages
- Awareness-focused web interface for learning cybersecurity concepts
- Interactive pages built for user learning

## Technologies Used

- HTML
- CSS
- JavaScript

## Project Purpose

The goal of this project is to help users understand:

- What phishing attacks are
- How attackers trick users into revealing sensitive information
- How to identify phishing attempts
- Best practices to stay safe online

## Learning Outcomes

Through this project I learned:

- Web development fundamentals
- Cybersecurity awareness principles
- How phishing attacks work
- Designing educational security tools
